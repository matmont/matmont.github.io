.Projects

ANSIA: A Neural System that Infers Affects

This application provides the visualization and saving of the Shimmer3's sensors data. Also, it implements a real-time stress detector with the use of Reservoir Computing Recurrent Neural Networks, based on the physiological sequences of the Shimmer3 sensors.

Shimmer3.py

This module provides a friendly interface to the Shimmer3 devices. It offers a set of API to handle the devices, for example to connect and disconnect them, to read data packet and to configurate the devices' settings.

TaskPopup.py

This custom Tkinter widget helps the management of blocking functions on a GUI environment. Long story short, it shows a loading window while executing the designed function.
